## Install Required Software

Download and install STM32CubeIDE from STMicroelectronics.
Operating System: Linux

## Set Up and Run the Project

Clone this repository or download the ZIP file:

```bash
$ git clone https://github.com/Silicon-Signals/stm32u5g9j-dk2-lvgl-benchmark.git
```

Open STM32CubeIDE, then go to:
File → Open Projects from File System → Directory,
select the cloned project folder, and click Finish.

Click the hammer icon to build the project (use the Debug configuration; it already has optimization enabled).

Click the play (Run) icon to flash the program to the board.
Once uploaded, the UI will appear on the board display.
